## Alternatives écartées (Veille comparée)
[23] Orpheus Project, « Orpheus-TTS: High-quality embedded text-to-speech engine », Orpheus Open Source, 2025. [En ligne]. Disponible sur : https://www.orpheus-tts.org/en. (Écarté de l'infrastructure finale car trop exigeant en ressources matérielles).

[24] vLLM Team, « vLLM: A high-throughput and memory-efficient LLM serving engine », GitHub, 2025. [En ligne]. Disponible sur : https://github.com/vllm-project/vllm. (Écarté : architecture trop lourde, optimisée pour la production scale-up).

[29] LibreChat Team, « LibreChat: Every AI Model in One App », LibreChat Docs, 2025. [En ligne]. Disponible sur : https://www.librechat.ai/. (Écarté : excellente interface utilisateur mais configuration multi-services trop lourde pour un PoC agile).
